Detailed Description

Search for computational zones lying in a shock wave. This function may be called if either ::SHOCK_FLATTENING has been set to MULTID or when ::ENTROPY_SWITCH has been turned on. Depending on the action to be taken, several flags may be switched on or off to control the update strategy in these critical regions.
